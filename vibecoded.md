# Vibecoded - Tool Documentation

This project was developed using AI-assisted vibe-coding tools to enhance productivity and UI design.

## Tools Used
- **Gemini CLI**: Used as the primary agent for code generation, architectural setup, and debugging.
- **Groq SDK**: Integrated for rapid AI model inference.

## Supporting Configurations
- The system prompt for Gemini CLI was configured to prioritize clean architecture, single-turn interactions as per the project requirements, and a minimalist dark-themed UI.
- `react-markdown` was used to ensure AI responses are formatted cleanly without manual CSS for every element.

## Vibe Palette
- **Warm Desert**: A professional yet cozy aesthetic using cream, beige, and soft amber to create a welcoming environment.
- **Amber & Terracotta**: High-contrast warm tones for primary actions and branding.

## Spirit
Cognitia AI is a refined, minimalist assistant that prioritizes user comfort and clarity through its warm, inviting interface.
